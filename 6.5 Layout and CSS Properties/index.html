CSS Box Model

The CSS box model describes the rectangular boxes generated for elements in the document tree and how they interact with each other.

This is a box with margin, padding, and border.

In the box model:

  • Content: The actual content of the box, where text and images appear.
  • Padding: The space between the content and the border. It is transparent and can be set to any size.
  • Border: A line that surrounds the padding (if any) and content. It can be styled with different widths, colors, and styles.
  • Margin: The space outside the border. It is also transparent and can be used to create space between elements.
